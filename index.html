Overview

Creating a server requires running the client in server mode from the Command Prompt. Follow the steps below to start a dedicated server instance.

Disclaimer

DISCLAIMER! In order for your friends to join, you will have to Port Forward. I'm not here to explain how to do that, just to make you aware how to set up a server. You can also use a P2P client like Radmin VPN or Hamachi VPN, or tools like Ngrok.
